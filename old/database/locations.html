NSW Locations

This pages provides up-to-date summary of all the known locations in NSW, basically an extract from my database. It is available in CSV format for ease of import into any spreadsheet program.

The extract contains the following columns:

LocationThe name of the location
LineThe name of the primary line the location lies on
SeqSequence number of the location along the line
TypeA code describing the main infrastructure at the location
StatusA code summarising the current state of the location
DistanceThe current distance in km from Central
Geo XThe longitude of the location (in GDA94 coords)
Geo YThe latitude of the location (in GDA94 coords)
ZoneThe AGD66 zone of the location
UTM XThe easting of the location (in AGD66/AMG coords)
UTM YThe northing of the location (in AGD66/AMG coords)
DescriptionA description of the location and its history
CurrentA description of the current state of the location
PhotosThe number of photographs of the location
DiagramsThe number of diagrams for the location
RefsThe number of references for the location

The data files:

  • locations.zip (Windows)
  • locations.csv.gz (Unix)
