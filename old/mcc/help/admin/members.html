Functions you are permitted to perform for members

Edit Members

You are presented with a hyperlinked list of member names. You may click on any of the names to find details of the member and as Administrator you are able to modify these parameters. You have complete control over members, including the ability to add or delete them. At the top left corner of the list you find the NEW hyperlink that you can use the create a new member.

Edit Groups

You are presented with a hyperlinked list of group names. You may click on any of the names to find details of the group and as Administrator you are able to modify these parameters.You may create or modify "Groups". Groups are special groups that you can create to restrict access to certain areas on your website. For example, you can create a area, where only the "test" group can access and see it. At the top left corner of the list you find the NEW hyperlink that you can use the create a new group. Before the group name you see an image hyperlink that can be used to mail all members of that group.

Manage requests

The members can request access to an other area, by using the hyperlinks of none authorized areas. You as an admistrator can grand or reject his request and make sure allrights are set with this one option.

E-mail Your Members

From this menu you can send messages to all members who've registered and entered their email addresses. You may edit the distribution list, or send messages to all. Useful for important update/news information. However, use sparingly, because the solution doesn't have an "opt in/out" feature yet, and may users may object to having messages sent to them without their authorization.

Upload Member Data

MCC provides the ability to upload members form an external source. Giving you the possiblity to migrate users form other systems. The file should contain for each member a line and the fields within the line are separated by a '|'. The sequence of fields is as follows: Name | Password | Full Name | E-Mail | Active (1/0=on/off) | Admin (1/0=on/off) | Group1+Group2 | Area1+Area2 | Flex Fields. Do not put spaces between the separators, here it's only done to make the helpfile readable. If a member should be added to more then one group or area use '+' as a separator between the items. If you are using the flexprofile options of MCC to extented the profile with additional fields, you can add the fields at the end of the line, each field separated by '|'. Be aware that no syntax checking is done and that lines starting with # are threaded as comments.

Remove Inactive Members

This options removes all inactive members that are inactive for the N number of days. N you have specified in the MCC configuration. Using this options together with inactive on registration will keep your memberlist correct and tidy.

Deactive non frequent members

Deactivate members that did not use you website for  N days and send them a e-mail with an activation code.
