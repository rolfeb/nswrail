How to use MCC in SMALL mode

In SMALL mode MCC only presents only a login screen and a list of availalble links. When it is combined with the direct mode options, it can also be use to show people a default page after they are logged in. Before you continue with this tutorial please be sure you understand the basics that which are explained in "What to do after installation". To start the tutorial click here.
