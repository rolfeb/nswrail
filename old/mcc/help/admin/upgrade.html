Let's Go For It!

 Upgrading from older version of MCC can be done in a view easy steps.

Steps that are REQUIRED:

  • Remove the content of the mcc/source directory
  • Upload the source directory of your distribution to mcc/source
  • Replace the mcc.pl and any language file with the ones in your distribution
  • Just to make it easy copy all images and helpfiles of your distribution

Steps that are optional:

  • If current version is 0.98 and you are using flexprofile change the separator | to ~ in your flexprofile.cfg
  • As of version 1.02 it is possible to have the data files in other directory then the root. Setting data directory to same value as root directory will keep the application working.
  • As of version 1.03 there is a addon directory. Add-ons have the extension .add and are all loaded from specified directory. Create the directory and change the settings.
  • As of version 1.06 there are database drivers. Default only the textfiles driver is available, if you have a license for a other driver download the newest and upload it to the source directory. When using perl_crypt as password protection please run the generate areas  from the admin page. This will apply the password fix to all your passwords.
  • As of version 1.08 a number of new images are added, so install don't forget to copy them over.

Congratulations!

Congratulations, you should be all set now. Enjoy your new version of MCC! =)
