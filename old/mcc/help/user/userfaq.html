Can only login

When a user logs in to MCC we store your username and password in a cookie. When there is no cookie we will keep asking you to login. Cookies are supported by almost all browser. Some browser or firewalls don't allow the usage of cookies by default you have to turn them on. See you browser helpfiles on how to do it.

Request access to other area

User can request access to other areas by clicking on the area for which they are not authorized. A request can be cancelled by using the link again. This option is only available when your site administrator has set this up.
