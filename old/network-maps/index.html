Network Maps

Snapshot Maps

The NSW network has grown (and shrunk!) over the last 140 odd years. These maps show the extent of the network for given years, and highlighting the changes for the previous 5 years.

The following snapshot maps are available:

  • The whole NSW network for a given year
  • The Sydney metropolitan network for a given year
  • The Newcastle/Maitland area network for a given year

Animation Maps

The above snapshot maps have been combined into an animation, showing the network changes over time.

The following animations are available:

  • The development of the whole NSW network over time
  • The development of the Sydney metropolitan network over time
  • The development of the Newcastle/Maitland area network over time
